<h1>SQL/R - Retail Sales Insights and Analysis</h1>

<h2>Description</h2>
This project investigates a dataset of retail sales that includes 200 individual transactions of customers based on multiple store locations, payment method, type of product purchased, and purchase amount.
<br />

<h2>Languages and Utilities Used</h2>

- <b>Rstudio</b> 
- <b>R programming language (for data visualization)</b>
- <b>SQLite (integrated in Rstudio)</b>

<h2>Environments Used </h2>

- <b>Windows 11</b>
